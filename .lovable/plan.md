
# Plano: Mudar a Cor de Fundo de Azul para Rosa

## Resumo
Alterar a cor de fundo principal do site de azul para rosa, mantendo a harmonia visual com os outros elementos.

---

## O Que Será Alterado

### Arquivo: `src/index.css`

**Mudança na linha 12:**
- **De:** `--background: 210 100% 50%;` (Azul)
- **Para:** `--background: 330 100% 60%;` (Rosa vibrante)

Esse valor HSL representa um rosa vibrante que combina bem com os elementos existentes do site (cards, botões, texto preto).

---

## Resultado Visual

O fundo do site inteiro mudará de azul para rosa, incluindo:
- Página inicial (Index)
- Página de autenticação (Auth)
- Todas as outras páginas

Os demais elementos (cards, textos, botões) permanecerão com suas cores atuais, criando um contraste harmonioso.

---

## Detalhes Técnicos

A cor rosa escolhida (`330 100% 60%`) em formato HSL significa:
- **H (Hue):** 330° - tom rosa
- **S (Saturation):** 100% - cor totalmente saturada
- **L (Lightness):** 60% - luminosidade média

Esta cor é vibrante e mantém boa legibilidade com o texto preto atual.
